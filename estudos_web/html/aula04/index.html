Tags com Link, ou Ancôragem.

Link para alguma pagina

O 'a href' contém um link, com isso podemos ir para outra aba, mas outro atributo da Tag seria o 'Target'. No qual com ele podemos definir se irá abrir em outra página ou na mesma página.

Temos outro atributo que seria o 'Rel', ele atribui a definição de qual a conexão com aquele link, seja 'Patrocínio', 'Relacionado' etc


Imagem e atributos

No caso da Imagem, usamos a Tag 'img' no qual atráves do 'src=' podemos conectar seja com uma imagem de algum caminho interno do seu computador, ou alguma da internet.

Dentro da Tag, também temos o atributo 'alt=' que seria justamente para descrever a imagem, que pode ser muito útil para leitores de tela (Usados por pessoas que tem necessidades especiais).

Ela também tem o atributo 'width' e o 'height', que são para ajuster do tamanho da imagem, no caso hoje em dia usamos muito mais o 'CSS' para essa funcionalidade, podemos resolver


Diferença entre e '< span >' para '< div >'.

O span Apenas troca estética. Já a div altera a semantica também. Então caso você queira destacar algo por apenas aparência mesmo, utilize 'span' se a mudança não for significativa para o usuário.

Citação e Itálico

Quando queremos fazer uma citação semantica, ao ínves de usarmos apenas as 'aspas', podemos usar a tag 'q' com o atributo 'cite' referênciando nossa citação.
Caso opte por um bloco separado do seu texto/blog temos a opção do 'blockquote' que é uma tag própria para separar a citação, já vem estilizada, mas podemos ajusta-la, e dentro delas podemos ter outras tags como 'p', 'h1'...

Aside


O elemento HTML representa uma seção de uma página que consiste de conteúdo que é tangencialmente relacionado ao conteúdo do seu entorno, que poderia ser considerado separado do conteúdo. Essas seções são, muitas vezes, representadas como barras laterais. Elas muitas vezes contem explicações laterais, como a definição de um glossário; conteúdo vagamente relacionado, como avisos; a biografia do autor; ou, em aplicações web, informações de perfil ou links de blogs relacionados.

---

Header(Introdução/Cabeçalho/Navegação princípal)

Header seria o Cabeçalho da página, pois o mesmo sempre estará no inicio, nele pode conter outras tags associadas ao menu princípal, pode conter também, a própria tag 'nav', para os menus de Navegação

  • Link1
  • Link2
  • Link3

Mas lembre-se, as tags não irão mudar muito no peso estético sem o CSS ou algum framework, se deseja melhorar é preciso dar esse próximo passo. Mas até então na semântica está tudo certo.


---

Main(Conteúdo princípal)

A tag 'main' representa o conteúdo princípal do 'body' da página, no caso ela deve ser uma tags única na semantica da página

---

Section(Sessões Genéricas)

Geralmente o section representa diversos trechos genéricos de um site quando não conseguimos especifica-lo melhor, por exemplo, em um blog, provavelmente vamos trabalhar com 'section' para separar diferentes blogs do mesmo conteúdo.

---

Footer(Rodapé)

O 'footer' seria o rodapé da página, pode estar tanto assiciado ao contéudo princípal, á uma mensão, á um bloco especifico, o importante é que ele é o rodapé daquele determinado conteúdo.

---

Glossário(Descrição de Termos)

HTML Definition List (dl)
Engloba uma lista de pares de termos e descrições. Um uso comum para este elemento é para implementar um glossário ou exibir metadados.
